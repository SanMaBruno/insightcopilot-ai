# Screenshots

Esta carpeta contiene las capturas de pantalla del proyecto para el README y documentación.

## Capturas esperadas

| Archivo | Descripción |
|---|---|
| `home.png` | Pantalla de inicio con capacidades del producto |
| `profile.png` | Vista de perfil estadístico de un dataset |
| `insights.png` | Insights clasificados por severidad |
| `charts.png` | Visualizaciones generadas automáticamente |
| `query.png` | Consulta analítica en lenguaje natural |
| `api-docs.png` | Documentación Swagger de la API |

## Cómo generar las capturas

1. Levantar backend y frontend (ver README principal).
2. Subir `data/sample/ecommerce_products.csv`.
3. Navegar por cada sección y tomar capturas de 1280×800 px.
4. Guardar aquí con los nombres indicados arriba.
5. Formato PNG, optimizadas para web (< 500 KB cada una).

## Para video demo

Guardar el video demo como `docs/demo-video.mp4` o agregar un link en el README principal.
